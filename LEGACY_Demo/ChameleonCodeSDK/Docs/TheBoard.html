The board has 9 cells. The meta data associated with these cells describes their...

  • Name
  • Description
  • X/Y Coordinates
  • An Index
  • An Index After a 90 degree clockwise rotation.
  • An Index after being flipped vertically

These are examples of things which are calculated (and included statically in this spec) which means that even though there are now 6 target platforms for this Tic-Tac-Toe implementation, none of them will need to actually calculate how to rotate a 9x9 grid 90 degrees - they can just follow the map established here, liked this.

The Cells

Each of the 9 cells can have one of these 3 cell states.

  1. You
  2. Them
  3. NoOne
