Google Resource Lists

There are 4 lists of resources defined in the google resources folder in this project. They are.

Descriptions

    Properties

      ExampleData

        AdditionalResources

        1. View Edit
          Settings
          Global settings for the entire TicTacToe project
        2. View Edit
          AdditionalResources
          A list of additional resources used by the project
        3. View Edit
          ProjectIntro
          An introduction to the project
        4. View Edit
          Entities
          A list of entities in the project (Player, Opponent, Game, Mark, Win, Loss, etc)
        5. View Edit
          LanguageTokens
          A list of language tokens used in the app.
        6. View Edit
          AILevels
          A list of AI Levels which the game supports starting with None (no ai) and working up to an AI that will win or tie every game.
        7. View Edit
          AIStrategies
          A list of strategies used by the AI, and which AI levels that start be used at.
        8. View Edit
          Translations
          A list of transaltions which any implementation of this game needs to support
        9. View Edit
          Cells
          A formal list of the cells in the 3x3 game of tic tac toe.
        10. View Edit
          CellPatterns
          A list of cells and a target if a match is found.
        11. View Edit
          CellStates
          A list of states that a cell can have
        12. View Edit
          TranslationTests
          These values validate the rotate and flip index values
        13. View Edit
          TargetPlatforms
          A list of target platforms that the SDK will support
