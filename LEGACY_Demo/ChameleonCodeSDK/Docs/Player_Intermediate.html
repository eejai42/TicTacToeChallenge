This player employes the following strategies.

Win If Possible
If any two cells are already filled in by the AI, and the 3rd cell in that row is available, play there.

Prevent Win If Possible
Prevent opponent from using WinIfPossible strategy

Force Win Next Turn
Check for a move that will force a win the next turn, regardless of what the opponent does.

Prevent Force Win Next Turn
Prevent opponent from using ForceWinNextTurn strategy

Play In Middle If Available
If there are no other winning strategies, play in the middle cell

Play In Corner If Available
If there are any corner positions available, player there

Random
Choice from a random selection amongst available cells
